# **App Name**: GymCentral

## Core Features:

- User Registration: Allow users to register with name, phone, and email. Store data in Firestore.
- Credential Upload: Enable users to upload a photo of their official credentials from their camera or gallery, and store in Firestore. Only a placeholder URL will be saved.
- Simulated Facial Registration: Simulate facial registration with a camera screen and oval guide. Display a 'Face captured' and 'Verification of life: OK (simulated)' message.
- Subscription Status: Display the user's subscription status (Active/Expired/Pending), remaining days, and expiration date.
- Staff Approval Workflow: Allow staff to approve/reject user registrations with a reason. Assign a membership plan (Monthly/Quarterly/Annual) and start date. Calculate expiration and remaining days.
- Subscription Management: Enable staff to activate/deactivate user subscriptions.
- Demo Mode Switch: Implement a switch to enter as 'User' or 'Staff'.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#F0F2F5) to maintain a clean and modern look.
- Accent color: Teal (#009688) for interactive elements and highlights.
- Body and headline font: 'PT Sans' for a modern and readable feel.
- Use sharp and modern icons to represent actions and statuses.
- Maintain a clean and structured layout to ensure a professional look and feel.
- Implement subtle animations to enhance user experience when loading data or transitioning between screens.
